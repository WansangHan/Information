# FlatBuffer

 **FlatBuffer**는 메시지 송/수신에 사용되는 플랫폼 종속성 없이 사용가능한 **"직/역직렬화 라이브러리"** 이다.

* ### 지원 언어
  * **C**
  * **C++**
  * **C#**
  * **Go**
  * **Java**
  * **Php**
  * **Pyhton**

* ### 플랫 버퍼를 쓰는 이유?
  * 데이터 송/수신시 **파싱/언패킹을 안해도 된다**.
  * **메모리 효율성이 높고, 빠른 속도를 보장한다.**
  * **유연성**(사용하는 **데이터 타입**에 대한)
  * **적은 양의 코드**로 작성 가능하다.
  * 사용하기 **편리**하다.
  * **크로스 플랫폼, 종속성 없이** 사용이 가능
